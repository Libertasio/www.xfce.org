Anmerkungen zu Xfce 4.2

Dies ist eine erste, improvisierte Seite mit Informationen zur aktuellen Entwicklerversion der Xfce Desktop Umgebung, die in hoffentlich nicht allzu ferner Zukunft in der Version 4.2 münden wird. Es handelt sich hierbei keinesfalls um einen präzisen Releaseplan, da bisher auch noch kein genaues Datum für das Erscheinen der Version 4.2 abzusehen ist. Die aktuelle, für Anwender empfohlene Version ist Xfce 4.0.5, welche von hier heruntergeladen werden kann.

Wie kann ich die neue Version testen?

Sie können die aktuelle Entwicklerversion aus unserem CVS Repository beziehen und installieren. Eine Anleitung hierzu ist auf dieser Seite verfügbar. Diese Version enthält naturgemäß etliche Fehler, unvollständige Übersetzungen, veraltete Dokumentation und es ist durchaus möglich, dass sie sich zu einem bestimmten Zeitpunkt garnicht erst kompilieren lässt. Sollten Ihnen derartige Fehler auffallen, benutzen Sie bitte das Xfce bug tracking system um uns von diesen Fehlern in Kenntnis zu setzen. Vergewissern Sie sich aber bitte zuvor, dass derselbe Fehler nicht bereits durch einen anderen Anwender gemeldet worden ist.

Xfce 4.2 enthält viele neue Features und einige neue Anwendungen.

Liste der Änderungen

Hier ist eine mehr oder weniger unvollständige Liste mit Änderungen, die für den Benutzer von Interesse sind:

Generell
  • Der Mehrbildschirmbetrieb wurde masgeblich verbessert.
  • support for partial struts in xfwm4, xftaskbar4 and xfce4-panel.
  • Die Behandlung von Menus und Icon Themes folgt nun der freedesktop Spezifikation
xftaskbar4
  • neue Option "Anwendungen immer gruppieren"
  • Ausrichtung einstellbar (links/rechts/zentriert)
  • Breite einstellbar
  • Ein Rechtsklick Menu wurde hinzugefügt
xfwm4
  • support for image composition with PNG images as layers, which means that one can do gradients or anything else, and that applies to any pixmap that make the window frame
  • etliche neue Fensterdekorationen
  • der Speicherort für Fensterdekorationen hat sich geändert
  • Unterstützung für das "show desktop" Protokoll
  • Unterstützung für KDE Kontext Hilfe
  • neue Option "Immer oben" für Fenster
  • icons in the "Alt+Tab" dialog
Settings manager
  • neue Optionen wie Schriften gerendert werden sollen
  • ein gemeinsamer Dialog für "Desktop workspaces" und "Workspace margins"
  • Monitorauflösung und Gamma Korrektur Dialog hinzugekommen
Neuer Sitzungs Manager
  • integrierte Kompatibilität mit KDE und Gnome
  • Mehrbildschirmbetrieb
  • Ist nicht auf smproxy (oder gnome-smproxy) angewiesen, sondern bietet einfache Unterstützung für Anwendungen, die nicht das X11R6 Session Management Protocoll implementieren.
  • bietet Unterstützung für schöne Themes
xfdesktop
  • Hintergrund kann nun auch ein Farbverlauf sein
  • Optionale Unterstützung für Icons in der Fensterliste und dem Arbeitsplatzmenu
  • Menueditor
  • Plugin für das Panel. Damit ist das Arbeitsplatzmenu nun auch über das Panel erreichbar.
xffm
  • Unterstützung für zwei Sichten in einem Fenster
  • Icon Theme Editor
  • viele neue Features
panel
  • der Einstellungsdialog wurde vereinfacht
  • neues Positionierungsschema
  • neue Plugin werden nun über einen Dialog dem Panel hinzugefügt.
  • prettier move handles
  • nicht benutzte Module werden nun automatisch entladen
xfce4-mixer
  • komplett neu geschrieben
xfce4-appfinder
  • neue Xfce4 Anwendung
gtk-xfce-theme-engine
  • neue Themes
