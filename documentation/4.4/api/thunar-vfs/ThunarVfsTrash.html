thunar_vfs_trash_get_files ()

GList*      thunar_vfs_trash_get_files      (ThunarVfsTrash *trash);

Returns the list of files stored within trash. The returned list contains only the basename of each file relative to the trashs files/ subdirectory.

The returned list and the string contained within the list are owned by trash and must not be freed by the caller.

trash : a ThunarVfsTrash.
Returns : the list of files stored within trash.

thunar_vfs_trash_get_info ()

ThunarVfsTrashInfo* thunar_vfs_trash_get_info
                                            (ThunarVfsTrash *trash,
                                             const gchar *file);

Determines the trash info for the given file in trash. The trash info currently contains the original path and deletion date of the file.

This function may return NULL if there's no trash info associated with file, even tho the file is listed as being stored within trash.

The caller is responsible for freeing the returned trash info object using the #thunar_vfs_trash_info_free() function.

trash : a ThunarVfsTrash.
file : the basename of the trashed file.
Returns : the trash info for file or NULL.

thunar_vfs_trash_get_info_path ()

gchar*      thunar_vfs_trash_get_info_path  (ThunarVfsTrash *trash,
                                             const gchar *file);

Determines the absolute path to the trash info file for file, which includes various informations stored when trashing the file.

This function should be rarely needed.

The caller is responsible to free the returned string using #g_free().

trash : a ThunarVfsTrash.
file : the basename of the trashed file.
Returns : the absolute path to the trash info file for file.

thunar_vfs_trash_get_uri ()

ThunarVfsURI* thunar_vfs_trash_get_uri      (ThunarVfsTrash *trash,
                                             const gchar *file);

Generates a 'trash://' URI that refers to the file in trash.

You'll need to call #thunar_vfs_uri_unref() on the returned object when you are done with it.

trash : a ThunarVfsTrash.
file : the basename of the trashed file.
Returns : the generated ThunarVfsURI.
