thunar_vfs_thumb_factory_can_thumbnail ()

gboolean    thunar_vfs_thumb_factory_can_thumbnail
                                            (ThunarVfsThumbFactory *factory,
                                             const ThunarVfsURI *uri,
                                             const ThunarVfsMimeInfo *mime_info,
                                             ThunarVfsFileTime mtime);

Checks if factory can generate a thumbnail for the file specified by uri, which is of type mime_info.

The usage of this method is thread-safe.

factory : a ThunarVfsThumbFactory.
uri : the ThunarVfsURI to a file.
mime_info : the ThunarVfsMimeInfo of the file referred to by uri.
mtime : the modification time of the file referred to by uri.
Returns : TRUE if factory can generate a thumbnail for uri.

gboolean    thunar_vfs_thumb_factory_has_failed_thumbnail
                                            (ThunarVfsThumbFactory *factory,
                                             const ThunarVfsURI *uri,
                                             ThunarVfsFileTime mtime);

Checks whether we know that factory won't be able to generate a thumbnail for the file referred to by uri, whose modification time is mtime.

The usage of this method is thread-safe.

factory : a ThunarVfsThumbFactory.
uri : the ThunarVfsURI to a file.
mtime : the modification time of the file referred to by uri.
Returns : TRUE if factory knows that it cannot generate a thumbnail for uri, else TRUE.

thunar_vfs_thumb_factory_generate_thumbnail ()

GdkPixbuf*  thunar_vfs_thumb_factory_generate_thumbnail
                                            (ThunarVfsThumbFactory *factory,
                                             const ThunarVfsURI *uri,
                                             const ThunarVfsMimeInfo *mime_info);

Tries to generate a thumbnail for the file referred to by uri in factory.

The caller is responsible to free the returned GdkPixbuf using g_object_unref() when no longer needed.

The usage of this method is thread-safe.

factory : a ThunarVfsThumbFactory.
uri : the ThunarVfsURI to the file for which a thumbnail should be generated.
mime_info : the ThunarVfsMimeInfo for the file referred to by uri.
Returns : the thumbnail for the uri or NULL.

thunar_vfs_thumb_factory_store_thumbnail ()

gboolean    thunar_vfs_thumb_factory_store_thumbnail
                                            (ThunarVfsThumbFactory *factory,
                                             const GdkPixbuf *pixbuf,
                                             const ThunarVfsURI *uri,
                                             ThunarVfsFileTime mtime,
                                             GError **error);

Stores pixbuf as thumbnail for uri in the right place, according to the size set for factory.

If you specify NULL for pixbuf, the factory will remember that the thumbnail generation for uri failed.

The usage of this method is thread-safe.

factory : a ThunarVfsThumbFactory.
pixbuf : the thumbnail GdkPixbuf to store or NULL to remember the thumbnail for uri as failed.
uri : the ThunarVfsURI of the original file.
mtime : the last known modification time of the original file.
error : return location for errors or NULL.
Returns : TRUE if the thumbnail was stored successfully, else FALSE.
