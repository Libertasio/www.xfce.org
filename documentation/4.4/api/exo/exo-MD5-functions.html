Description

This is an implementation 16-byte MD5 hash algorithm with some related utility functions.

Details

ExoMd5Digest

typedef struct {
  guchar digest[16];
} ExoMd5Digest;

Structure to represent a MD5 digest.

guchar digest[16]; the MD5 digest.
